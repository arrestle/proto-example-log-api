# Examples

| Directory | Language | Purpose |
|-----------|----------|---------|
| `go/` | Go | Basic logging (minimal) |
| `py/` | Python | Basic logging (minimal) |
| `go_otlp/` | Go | OpenTelemetry OTLP format logging |
| `py_google/` | Python | Logging with Google MessageToDict library |
| `go_gateway/` | Go | Gateway User API demonstration |
| `py_gateway/` | Python | Gateway User API demonstration |
| `go_awx/` | Go | AWX Job Template API demonstration |
| `py_awx/` | Python | AWX Job Template API demonstration |

**Run all examples:** `make run-all`
